# ⏰ Digital Clock

A simple and responsive **Digital Clock** built using **HTML, CSS, and JavaScript**.
It displays the current time in real-time and updates every second.

---

## 🚀 Features

* 🕒 Real-time clock (updates every second)
* 🎨 Clean and modern UI
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Lightweight and fast

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript 

---

## 💡 How It Works

* JavaScript uses `Date()` object to get current time
* `setInterval()` updates time every second
* DOM manipulation is used to display time dynamically
